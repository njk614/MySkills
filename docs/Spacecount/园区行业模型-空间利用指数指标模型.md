## 1 引言

### 1.1 目的与业务价值

空间利用指数是衡量园区空间资源利用水平的综合战略指标。依据国家《党政机关办公用房建设标准》、GB/T 50908-2013《绿色办公建筑评价标准》及“三生融合”的园区发展理念，通过对空间活力（占用率与使用频次）、空间集约度（人均面积与空间复用）、空间品质（履约率与冲突解决）的精细化统计与聚合，形成可量化、可追溯的空间利用评价体系，为空间优化、成本控制、资源调配提供决策依据。

**业务价值**：

- **实时监控空间占用情况**：识别闲置工位与会议室，支撑灵活办公策略。
- **量化空间利用效率**：分析人均面积达标率与空间复用频次，为空间规划提供数据支撑。
- **优化预定与冲突管理**：监控履约率与冲突解决率，提升空间调度效率。
- **数据驱动空间优化**：实现从“被动分配”到“主动释放”的转变。

### 1.2 适用范围

本文档适用于孪易平台中办公楼层空间管理的指标计算，涵盖固定工位、灵活工位、会议室、工位感应器、人体红外传感器、空间预定记录等数据源。空间划分为7个业务区域：

| 区域名称      | 区域类型  | 包含内容             | 是否参与空间利用指标         |
|:--------- |:----- |:---------------- |:------------------ |
| **固定工位区** | 固定工位区 | 销售部、研发部、设计部等固定工位 | 是                  |
| **灵活工位区** | 灵活工位区 | 共享工位、临时工位        | 是                  |
| **接待区**   | 接待区   | 前台、接待沙发区（含固定工位）  | 是                  |
| **办公一区**  | 办公一区  | 项目部、运营部固定工位      | 是                  |
| **办公二区**  | 办公二区  | 市场部、产品部固定工位      | 是                  |
| **会议室区**  | 会议室区  | 大会议室、小会议室        | 是                  |
| **机房**    | 机房    | 服务器、网络设备         | 否（仅设备监控，不参与空间利用评价） |

注：机房、配电间等设备区域无人员长期占用，不纳入空间利用指数评价体系，其面积不计入整层加权计算的分母。

### 1.3 核心设计原则

- **从下到上**：所有L3及以上指标均基于孪生体数据模型计算，确保数据可追溯、可落地。
- **孪易职责**：孪易数据中台负责接入原始数据，并完成从原始数据到L1/L2/L3指标的完整计算，为应用层提供可直接消费的指标结果。
- **空间定位**：通过设备台账中的“所属区域”字段将设备与物理办公区域关联，实现区域级聚合；整层级指标由区域级指标按面积加权平均得到。
- **时间聚合**：小时值为基础粒度（滑动窗口3600秒，每分钟更新），日值由小时值平均得到，周值由日值平均得到，月值由日值平均得到。**所有L3、L2指标均支持小时、日、周粒度；L1指标支持小时、日、周、月粒度。**
- **数据粒度**：所有传感器时序原始数据粒度为“秒”，派生字段基于滑动窗口3600秒计算，每分钟更新一次。
- **数据有效性阈值**：采用分层处理规则：
  - **单个空间**：若该空间（工位/会议室）对应的传感器在线率低于80%，则该空间不计入区域聚合的**分母**，其占用率、使用次数等也不参与平均（即剔除）。
  - **区域级**：若区域内某类传感器有效空间数 = 0（即所有该类空间均因低在线被剔除），则该区域该类相关的所有L3指标直接计为0分。
  - **整层**：整层聚合时，只计入有效区域（区域指标有效），按面积加权平均。若所有有效区域数 = 0，则整层指标计为0分。
- **工作时段过滤**：空间利用相关指标仅统计工作时段（如9:30-19:30），非工作时段不计入日/周/月平均值。时间边界采用左闭右开：09:30:00 ≤ 时间 < 19:30:00。跨时段事件按事件发生时间归属，不跨时段拆分。跨天预定按预定开始时间所在日期归属。
- **指标依据**：每个L3指标必须引用具体的国际、国家或行业标准条款，不得使用“企业标准”等无权威来源的表述。
- **指标方向**：所有L3、L2、L1指标均设计为**越高越好**，便于直观理解。

## 2 指标体系层级图

```
L1: 空间利用指数
│
├─ L2: 空间活力
│   ├─ L3: 空间占用率
│   └─ L3: 空间使用频次
│
├─ L2: 空间集约度
│   ├─ L3: 人均面积达标率
│   └─ L3: 空间复用指数
│
└─ L2: 空间品质
    ├─ L3: 灵活工位预定履约率
    ├─ L3: 会议室空置率
    └─ L3: 空间冲突解决率
```

**说明**：L3指标中，部分指标（人均面积达标率、空间复用指数、灵活工位预定履约率、会议室空置率、空间冲突解决率）因业务定义本身不适合小时粒度（如人均面积以天为单位统计），故只支持日/周粒度。L2指标基于L3计算，对于依赖日/周粒度L3的L2子指标，其小时值通过将日值回填到该日各小时实现（即小时值等于该日日值），以保证L2整体支持小时粒度，便于统一展示和下钻。

**空间状态计算说明**：

- 该字段为区域级派生字段，由计算中台每分钟基于预定系统数据和占用状态计算一次。
- 预定状态：通过查询空间预定记录，判断当前时刻该灵活工位是否处于已预定时段内。
- 占用状态：由工位感应器上报的“占用状态”提供。
- 组合规则：
  - 未被预定 + 无人 → 空闲可用
  - 未被预定 + 有人 → 临时使用
  - 已被预定 + 无人 → 预定空置
  - 已被预定 + 有人 → 预定使用

### 3.4 数据接入与计算规范

- **传感器原始数据接入**：通过物联网网关每秒推送原始数据至孪易，格式包含时间、设备ID及各原始值。
- **传感器级派生计算**：孪易计算模块每分钟对每个传感器执行滑动窗口统计（过去3600秒），生成在线率、使用次数（状态变化次数）、恢复时间、占用率_1小时等派生字段，更新到该传感器的时序记录中。
- **区域级派生计算**：孪易计算模块每分钟对每个区域执行聚合计算：
  - 区域占用状态：取区域内所有工位/会议室占用状态的最大值（任一占用即为占用）。
  - 区域占用率_1小时：区域内所有有效工位/会议室的平均占用率（按数量平均），仅计入在线率≥80%的空间。
  - 空间状态（灵活工位和会议室）：结合预定系统数据（判断当前时刻是否在已预定时段）和占用状态，计算四种状态值。
- **数据对象接入**：空间预定记录通过API实时推送，每条记录包含所有字段，孪易负责存储并提供查询。
- **数据有效性分层处理**：
  - **单空间**：若某空间（工位/会议室）对应的传感器在线率<80%，该空间不计入区域聚合的**分母**，其占用率、使用次数等不参与平均。
  - **区域**：若区域内某类传感器有效空间数=0（所有该类空间均被剔除），则该区域该类相关的所有L3指标直接计为0分。
  - **整层**：整层聚合时，只计入有效区域（区域指标有效），按面积加权平均。若所有有效区域数=0，则整层指标计为0分。
- **工作时段过滤**：所有L3指标仅计算工作时段（默认9:30-19:30）的数据。时间边界采用左闭右开：09:30:00 ≤ 时间 < 19:30:00。非工作时段的数据不参与任何指标计算。日/周/月平均值基于工作时段的小时值计算。跨时段事件按事件发生时间归属，跨天预定按预定开始时间所在日期归属。

### 8.5 得分分析

| 区域     | 空间利用指数    | 主要短板                                                |
|:------ |:--------- |:--------------------------------------------------- |
| 固定工位区  | 49.9分     | 空间活力偏低（使用频次低），复用指数低                                 |
| 灵活工位区  | 59.8分     | 品质偏低（履约率仅66.7%）                                     |
| 接待区    | 53.3分     | 集约度偏低（人均面积过大）                                       |
| 办公一区   | 49.2分     | 活力偏低，复用指数低                                          |

| 会议室区   | 66.0分     | 品质一般（空置率和冲突解决率尚可）                                   |
| **整层** | **53.8分** | **整体空间利用率中等偏低，主要瓶颈在于固定工位区和办公区域的复用指数低，以及接待区人均面积过大。** |

**建议策略**：

1. 提高固定工位区、办公一区、办公二区的空间使用频次：鼓励员工多使用固定工位，可结合考勤数据分析闲置工位，考虑推行灵活办公或共享工位策略。
2. 优化灵活工位预定规则：降低预定未使用比例，可缩短免费保留时间或增加违约惩罚。
3. 接待区人均面积过大，可考虑缩小接待区面积或增加临时工位，提高空间集约度。
4. 会议室空置率已较高，可考虑释放部分会议室或调整预定规则。

### 9.5 术语表

| 术语            | 解释                                                   |
|:------------- |:---------------------------------------------------- |
| **空间占用率**     | 在统计周期内，工位和会议室被占用的平均时间比例。                             |
| **空间使用频次**    | 每个空间（工位+会议室）平均被使用的次数，使用定义为“从空闲变为占用”的一次变化。            |
| **人均面积达标率**   | 实际人均面积与标准人均面积的符合程度，反映空间利用的集约性。                       |
| **空间复用指数**    | 有工位的区域内，工位被使用的平均频次，反映空间复用效率。                         |
| **灵活工位预定履约率** | 预定后被实际使用的比例，反映用户行为的规范性。                              |
| **会议室空置率**    | 预定但未使用的比例，反映资源浪费程度。                                  |
| **空间冲突解决率**   | 预定冲突被解决的比例，反映管理效率。                                   |
| **同时段最大占用人数** | 工作时段内，工位占用状态为1的最大并发数，用于估算实际在岗人数。                     |
| **空间状态**      | 灵活工位和会议室的实时状态，由预定状态和占用状态组合而成，取值为空闲可用、临时使用、预定空置、预定使用。 |
| **有效空间**      | 在线率≥80%的空间，参与指标聚合计算。                                 |
| **有效区域**      | 区域内至少有一类传感器的有效空间数>0，其区域指标有效。                         |
| **可配置参数**     | 支持系统动态调整的参数，包括权重、基准值、标准值、阈值等。                        |
| **工作时段**      | 默认9:30-19:30，采用左闭右开边界，可在系统中配置。                       |
